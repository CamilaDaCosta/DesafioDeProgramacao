# :rocket:Desafio De Programação Academia Capgemini:rocket:

Projetos desenvolvidos na IDE IntelliJ Community Edition. <br />
Utilizando Linguagem Java. <br /><hr>

### [QUESTÃO #1](https://github.com/CamilaDaCosta/DesafioDeProgramacao/blob/master/Questao1/src/Main.java) <br />
Escreva um algoritmo que mostre na tela uma escada de tamanho n utilizando o caractere * e espaços. 
A base e altura da escada devem ser iguais ao valor de n. A última linha não deve conter nenhum espaço.

### [QUESTÃO #2](https://github.com/CamilaDaCosta/DesafioDeProgramacao/blob/master/Questao2/src/Main.java) <br />
Verificar se a senha informada segue os requisitos de segurança informados (detalhados no comentário inicial do código).

### [QUESTÃO #3](https://github.com/CamilaDaCosta/DesafioDeProgramacao/blob/master/Questao3/src/Main.java) <br />
Duas palavras podem ser consideradas anagramas de si mesmas se as letras de uma palavra podem ser realocadas para formar a outra palavra. 
Dada uma string qualquer, desenvolva um algoritmo que encontre o número de pares de substrings que são anagramas.
